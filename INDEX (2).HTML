The Main Language of the Web

HTML

HTML is the standard markup language for creating web pages.
HTML describes the structure of a web page, and consists of a series of elements.
HTML elements tell the browser how to display the content.

CSS

CSS is a language that describes how HTML elements are to be displayed screen, paper or in other media.
CSS saves a lot of work, because it can control the layout of multip web pages all at once.

Javascript

Jvascript is the programming language of HTML and the Web.
Javascript can change HTML content and attribute values.
Javascript can change CSS.
Javascript can hide and show HTML elements,and more.
